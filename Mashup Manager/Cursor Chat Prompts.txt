TIPS FOR BETTER RESULTS
================================================================================

- Be specific about data shapes — paste the TypeScript interface directly.
- Name the exact IDS components — Cursor will look them up in the library.
- Mention file paths of existing files you want to modify.
- Reference a specific example to reuse:
    "Use the same pattern as Mashup Manager/Projects Examples/Quick-Price-Quote/
     src/app/features/home/components/search/search/search.component.ts"
- One feature per prompt — break large mashups into multiple focused prompts.
- If unsure of an M3 program/transaction, just describe what data you need
  and let Cursor suggest the right MI call based on the project examples.
